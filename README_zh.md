## 项目简介

SafeColors 是一款创新的地址安全验证工具，通过将加密算法给用户地址增添独特颜色组合的视觉标识系统，有效防范以下风险：
- ✅ 相似地址钓鱼攻击
- ✅ 人工校验失误
- ✅ 恶意地址替换
